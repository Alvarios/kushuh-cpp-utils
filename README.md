# Kushuh's C++ Utils

## Installation

*From your project folder in terminal*

`git clone https://github.com/Alvarios/kushuh-cpp-utils ./path/to/utils/folder`

Then import .h files in your files :

`#include "./path/to/utils/folder/category/some_util.h"`
